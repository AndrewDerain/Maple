# 查看版本信息

使用参数 `-v`
```bash
    $ maple -v
```
输出如下
```bash
    $ maple -v
    maple version 0.0.0.0
    
    Maple Framework maintained and supported by: 
    AndrewDerain (https://github.com/AndrewDerain). 
    GitHub: https://github.com/AndrewDerain/Maple 
    E-mail: locomoco@yeah.net 
```


或是使用参数 `--version`
```bash
    $ maple --version
    maple version 0.0.0.0
    
    Maple Framework maintained and supported by: 
    AndrewDerain (https://github.com/AndrewDerain). 
    GitHub: https://github.com/AndrewDerain/Maple 
    E-mail: locomoco@yeah.net 
```
