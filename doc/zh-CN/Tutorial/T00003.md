# 查看版本信息

使用参数 `-v`
```bash
    $ fantasia -v
```
输出如下
```bash
    $ fantasia -v
    fantasia version 0.0.0.0
    
    Fantasia Framework maintained and supported by: 
    AndrewDerain (https://github.com/AndrewDerain). 
    GitHub: https://github.com/AndrewDerain/Fantasia 
    E-mail: locomoco@yeah.net 
```


或是使用参数 `--version`
```bash
    $ fantasia --version
    fantasia version 0.0.0.0
    
    Fantasia Framework maintained and supported by: 
    AndrewDerain (https://github.com/AndrewDerain). 
    GitHub: https://github.com/AndrewDerain/Fantasia 
    E-mail: locomoco@yeah.net 
```
