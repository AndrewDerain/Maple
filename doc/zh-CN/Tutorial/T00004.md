# 查看帮助信息


使用参数 `-h`
```bash
    $ maple -h
```

输出如下
```bash
    $ maple -h
                                                
    --------- maple usage manual ---------   
                                                
    Usage                                       
        maple [Options]                      
        maple [Command] [Command Options]    
                                                                    
    Options                                                         
        --version, -v           = Print version of framework.       
        --help, -h              = Print usage information.          
                                                                    
    Commands                                                        
        new                     = Create new project from template. 
          --template, -t <name> -> Indicate project template name.  
          --name, -n <name>     -> Set the name of your project.    

```

或是使用参数 `--help`
```bash
    $ maple --help
    
    --------- maple usage manual ---------   
                                                
    Usage                                       
        maple [Options]                      
        maple [Command] [Command Options]    
                                                                    
    Options                                                         
        --version, -v           = Print version of framework.       
        --help, -h              = Print usage information.          
                                                                    
    Commands                                                        
        new                     = Create new project from template. 
          --template, -t <name> -> Indicate project template name.  
          --name, -n <name>     -> Set the name of your project.    

```
